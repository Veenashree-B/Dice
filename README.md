# Dice Roller App 🎲

A fun and interactive Android application that simulates rolling a six-sided die. This project demonstrates foundational skills in building dynamic, state-driven user interfaces using modern Android development practices.

## ✨ Features
* **Interactive UI:** Allows users to "roll" a die with a tap of a button, instantly updating the screen with a random result.
* **Declarative UI Architecture:** Built entirely with **Jetpack Compose**, showcasing a clean and modern approach to structuring Android application interfaces.
* **State Management:** Effectively utilizes Compose state handling (`remember`, `mutableIntStateOf`) to dynamically update the dice image based on user interaction.
* **Efficient Resource Management:** Demonstrates the proper integration of vector graphics (`R.drawable.dice_1` through `dice_6`) and string resources within the Compose ecosystem.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Design System:** Material Design 3
* **Build System:** Gradle (Version Catalog enabled)
